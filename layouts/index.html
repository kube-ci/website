KubeCI

Kubernetes native CI system

Quick installation

Download pre-built binaries from kubeci/cli Github releases and put the binary to some directory in your `PATH`. Learn more

  • Linux
  • Mac
  • Helm Chart

$ curl -fsSL https://raw.githubusercontent.com/kubeci/cli/0.1.0/hack/deploy/kubeci.sh| bash

$ wget -O kubeci https://github.com/kube-ci/docs/releases/download/0.1.0/kubeci-linux-amd64 \
    && chmod +x kubeci \
    && sudo mv kubeci /usr/local/bin/

$ curl -fsSL https://raw.githubusercontent.com/kubeci/cli/0.1.0/hack/deploy/kubeci.sh| bash

$ wget -O kubeci https://github.com/kube-ci/docs/releases/download/0.1.0/kubeci-darwin-amd64 \
    && chmod +x kubeci \
    && sudo mv kubeci /usr/local/bin/

$ helm repo add appscode https://charts.appscode.com/stable/
$ helm repo update
$ helm install appscode/kubeci --name kubeci-operator --version 0.1.0 \
  --namespace kube-system
$ helm install appscode/kubeci-catalog --name kubeci-catalog --version 0.1.0 \
  --namespace kube-system
